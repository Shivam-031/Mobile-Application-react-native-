# 📱 Green Yatra India — Native Setup Guide

The `android/` and `ios/` folders are NOT included in this repo.
They are generated by React Native CLI. Follow these steps to create them.

---

## Why They're Not Included

React Native native folders contain gigabytes of compiled C++, Gradle,
and Xcode build files. They are always generated locally and never
committed to git — this is standard React Native practice.

---

## ✅ One-Time Setup (do this once per machine)

### Step 1 — Prerequisites

**All platforms:**
- Node.js 18+ installed
- Java 17+ installed (for Android)
- Android Studio installed with Android SDK

**Mac only (for iOS):**
- Xcode 14+ installed from Mac App Store
- CocoaPods installed: `sudo gem install cocoapods`

---

### Step 2 — Generate Native Folders

Run this from OUTSIDE the `mobile/` folder:

```bash
# Go up one level from mobile/
cd ..   # now you're in GreenYatraIndia_Production/

# Generate a fresh React Native project (same version as package.json)
npx react-native@0.73.0 init GreenYatraIndia --version 0.73.0

# Copy the generated native folders into mobile/
cp -r GreenYatraIndia/android  mobile/android
cp -r GreenYatraIndia/ios      mobile/ios

# Clean up the temp project
rm -rf GreenYatraIndia
```

---

### Step 3 — Fix App Display Name

Edit `mobile/android/app/src/main/res/values/strings.xml`:
```xml
<resources>
    <string name="app_name">Green Yatra India</string>
</resources>
```

Edit `mobile/ios/GreenYatraIndia/Info.plist` — find and update:
```xml
<key>CFBundleDisplayName</key>
<string>Green Yatra India</string>
```

---

### Step 4 — Set Up Firebase (for push notifications)

1. Go to https://console.firebase.google.com
2. Create a project called "green-yatra-india"
3. Add an Android app with package name: `com.greenyatraindia`
4. Download `google-services.json` → place at `mobile/android/app/google-services.json`
5. Add an iOS app with bundle ID: `com.greenyatraindia`
6. Download `GoogleService-Info.plist` → place at `mobile/ios/GreenYatraIndia/GoogleService-Info.plist`

---

### Step 5 — Install All Dependencies

```bash
cd mobile

# Install JavaScript dependencies
npm install

# Android: no extra steps needed — Gradle handles native deps

# iOS only (Mac):
cd ios
pod install
cd ..
```

---

### Step 6 — Run the App

**Android:**
```bash
# Start Metro bundler (keep this running)
npx react-native start

# In a new terminal, run on Android
npx react-native run-android
```

**iOS (Mac only):**
```bash
npx react-native run-ios
```

**Physical Android device:**
1. Enable Developer Options on your phone
2. Enable USB Debugging
3. Connect via USB
4. Run `adb devices` to confirm it's detected
5. Run `npx react-native run-android`

## 🔧 Common Issues

### "SDK location not found"
Create `mobile/android/local.properties`:
```
sdk.dir=/Users/YOUR_USERNAME/Library/Android/sdk      # Mac
sdk.dir=C:\\Users\\YOUR_USERNAME\\AppData\\Local\\Android\\Sdk  # Windows
```

### "Unable to load script from assets"
```bash
npx react-native start --reset-cache
```

### "Execution failed for task :app:processDebugGoogleServices"
Make sure `google-services.json` is at `android/app/google-services.json`

### Metro bundler port conflict
```bash
npx react-native start --port 8082
# Then in another terminal:
npx react-native run-android --port 8082
```

### iOS pod install fails
```bash
cd ios
pod deintegrate
pod install
```

## 📦 Android Release Build (APK)

```bash
cd android
./gradlew assembleRelease
# APK at: android/app/build/outputs/apk/release/app-release.apk
```

## 📦 Android Play Store Build (AAB)

```bash
cd android
./gradlew bundleRelease
# AAB at: android/app/build/outputs/bundle/release/app-release.aab
```

## 🍎 iOS Release Build

Open `ios/GreenYatraIndia.xcworkspace` in Xcode
→ Product → Archive → Distribute App → App Store Connect
